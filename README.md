# Project Name

skyView

## Dev setup

### setup python 3.12
- we are working in python 3.12

### Requirements
- pip install -r requirements.txt

### Run

To run the project on Windows, execute:

```bash
python main.py
```

To run the project on Linux, execute:

```bash
python3 main.py
```
### Build

To build the project using PyInstaller, execute:

```bash
pyinstaller main.spec
```
